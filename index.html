Hagfish - Creative use of dotplots in NGS data analysis

Hagfish is a tool that is to be used in data analysis of Next
Generation Sequencing (NGS) experiments. Hagfish builds on the concept
of coverage plots and aims to assist in:

  • quality control of de novo genome assembly
  • identification of structural variation in a genome re-sequencing
    experiment

The use of Hagfish requires a reference sequence and a paired end
re-sequencing data set. Hagfish has more power the larger the insert
size of the paired end library is.
